What this mockup is encoding

  • Titles, not counts — each node is the step title, so branching never makes a "3 of 7" lurch.
  • Repeatable = a branch line — "Dependents" drops into numbered instances (1, 2, + Add another). Here it's expanded because you're inside it; when you're elsewhere it collapses to a single node with a count.
  • Conditional steps — "Income" is greyed with an "appears if employed" tag; it slides in/out with a mini animation when the answer changes.
  • Three states — done (green, clickable), current (blue), locked (grey, not clickable — no skip-ahead).
  • Hidden on Review — the map disappears on the check-your-answers page (that page is the overview).
